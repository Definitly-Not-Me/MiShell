  **MiShell** is a very simple and dumb shell. It got some builtins commands
and ... that's all. I plan to improve it later (if I find time).

---

### Getting started:
1. Clone the repo and cd into it:
```bash
git clone https://github.com/Definitly-Not-Me/MiShell.git && cd MiShell/src/
```

2. Cool nixers and chad C devs will know how to simply compile it with:
```bash
gcc main.c logger.c -o MiShell
```
3. Run:
```
./MiShell
```
